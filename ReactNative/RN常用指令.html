RN常用指令

RN常用指令

react-native --v //查看当前reactNative版本 sudo npm update -g react-native-cli //更新全局rn到最新版本 npm info react-native //查看服务器端rn各版本的信息 npm install // 根据配置的package.json给下载的工程项目下载安装rn环境 react-native upgrade // 根据package.json配置的rn版本，更新rn环境代码 react-native run-ios // 运行ios工程 react-native run-android // 运行android工程 react-native init 工程名 // 初始化一个工程，下载react-native 的所有源代码和依赖包 npm install --save react-native @0.18 // 项目降级或升级到指定版本，记得之后要运行一下react-native upgrade 更新一下项目依赖等 npm install react-native-storage --save // 安装某个lib到项目中
